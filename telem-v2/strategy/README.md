# Live Strategy Lookup

This folder contains the firmware half of the telemetry strategy bridge.

- `strategy_indy.h` is generated by the telemetry software repo from an optimized dashboard strategy.
- `strategy_lookup.h/.cpp` maps live GPS latitude/longitude to the nearest strategy point.
- `getRecommendedSpeed(latitude, longitude)` returns target speed, action, segment, and distance from the exported track line.

The lookup is intentionally simple: the Python tools do the heavy optimization offline, and firmware only performs a nearest-track lookup with a local search window plus full-scan fallback.
